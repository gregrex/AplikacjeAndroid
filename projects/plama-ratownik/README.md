# Plama Ratownik

## Cel aplikacji

Aplikacja pomaga użytkownikowi szybko dobrać bezpieczną procedurę usuwania plamy z ubrania lub materiału.

MVP działa bez AI:

```text
typ plamy + materiał + świeżość plamy
→ reguła lokalna
→ wynik podstawowy
→ reklama rewarded
→ pełna instrukcja
```

## Grupa docelowa

- osoby piorące ubrania w domu,
- rodzice,
- studenci,
- osoby korzystające z Vinted/OLX,
- osoby chcące szybko uratować ubranie.

## Wartość dla użytkownika

- czego nie robić,
- pierwszy krok ratunkowy,
- pełna procedura po reklamie,
- lista potrzebnych środków,
- wariant awaryjny.

## Monetyzacja

- darmowy wynik podstawowy,
- pełny wynik po reklamie rewarded,
- opcjonalne premium bez reklam,
- opcjonalny eksport checklisty.

## Zakres MVP

- 10 typów plam,
- 6 typów materiałów,
- co najmniej 20 reguł,
- język PL na start,
- struktura pod EN i UK,
- historia i ulubione przez shared-engine.
