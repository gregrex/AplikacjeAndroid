# Testy manualne — Plama Ratownik

## Test 1 — start aplikacji

1. Uruchom aplikację.
2. Wybierz język polski.
3. Sprawdź, czy ekran główny pokazuje nazwę Plama Ratownik.

Oczekiwany wynik: aplikacja uruchamia się bez serwera i pokazuje ekran startowy.

## Test 2 — kawa na bawełnie

1. Wybierz kategorię Kawa.
2. Wybierz materiał Bawełna.
3. Wybierz, że plama jest świeża.
4. Przejdź do wyniku.

Oczekiwany wynik: aplikacja pokazuje wynik podstawowy dla kawy na bawełnie.

## Test 3 — odblokowanie pełnego wyniku

1. Wykonaj Test 2.
2. Kliknij Odblokuj pełną instrukcję.
3. W trybie mock reklamy potwierdź odblokowanie.

Oczekiwany wynik: aplikacja pokazuje pełną instrukcję.

## Test 4 — krew świeża

1. Wybierz kategorię Krew.
2. Wybierz dowolny materiał.
3. Wybierz, że plama jest świeża.

Oczekiwany wynik: aplikacja pokazuje ostrzeżenie o unikaniu gorącej wody.

## Test 5 — wynik domyślny

1. Wybierz kategorię bez szczegółowej reguły.
2. Przejdź formularz.

Oczekiwany wynik: aplikacja pokazuje ogólną procedurę usuwania plamy.

## Test 6 — historia

1. Wygeneruj wynik.
2. Zapisz lub przejdź do historii.

Oczekiwany wynik: ostatni wynik jest widoczny w historii.

## Test 7 — ulubione

1. Wygeneruj pełny wynik.
2. Dodaj do ulubionych.
3. Otwórz ekran Ulubione.

Oczekiwany wynik: wynik jest widoczny na liście ulubionych.
