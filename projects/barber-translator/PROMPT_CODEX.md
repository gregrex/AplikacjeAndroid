# Prompt dla Codexa — Barber Translator

## Cel

Przygotuj dane projektu `barber-translator` dla wspólnego silnika aplikacji Android.

## MVP

Użytkownik odpowiada na pytania o fryzurę, a aplikacja tworzy neutralną instrukcję dla fryzjera.

## Dane do przygotowania

- typy włosów
- długości
- style boków
- style góry
- poziomy formalności
- szablony instrukcji
- lista czego nie skracać

## Zasady

Nie oceniaj twarzy ani atrakcyjności. Nie generuj porad medycznych dotyczących skóry głowy. Projekt ma być praktycznym tłumaczem preferencji.
