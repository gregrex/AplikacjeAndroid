# Barber Translator

## Cel aplikacji

Aplikacja pomaga użytkownikowi przygotować prostą instrukcję dla fryzjera lub barbera.

## Schemat MVP

typ włosów -> długość góry -> boki -> styl -> instrukcja podstawowa -> reklama -> warianty instrukcji

## Zakres MVP

Parametry:

- długość góry
- krótkość boków
- typ przejścia
- grzywka
- tekstura
- styl naturalny lub elegancki

## Wynik

Aplikacja generuje krótki opis do pokazania fryzjerowi.

## Monetyzacja

- jedna instrukcja za darmo
- trzy warianty po reklamie
- wersja do zapisania jako obraz po reklamie
- wersja bez reklam

## Zasady

Aplikacja nie ocenia wyglądu osoby. Pomaga tylko opisać preferencje dotyczące fryzury.
