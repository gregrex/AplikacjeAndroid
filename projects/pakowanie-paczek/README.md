# Pakowanie Paczek

## Cel aplikacji

Aplikacja pomaga dobrać sposób zabezpieczenia przedmiotu do wysyłki.

## Schemat MVP

typ przedmiotu -> kruchość -> rozmiar -> sposób transportu -> instrukcja podstawowa -> reklama -> pełna checklista

## Kategorie startowe

- ubrania
- książki
- szkło
- elektronika
- zabawki
- kosmetyki
- buty
- mały sprzęt

## Wynik

Aplikacja podaje:

- jak zabezpieczyć przedmiot
- jakie materiały przygotować
- czego unikać
- checklistę przed nadaniem

## Monetyzacja

- instrukcja podstawowa za darmo
- pełna checklista po reklamie
- wariant ekonomiczny i bezpieczny po reklamie

## Zasady

Aplikacja nie gwarantuje bezpieczeństwa przesyłki. Dla bardzo wartościowych lub delikatnych przedmiotów zaleca dodatkowe ubezpieczenie i ostrożność.
