# Prompt dla Codexa — Pakowanie Paczek

## Cel

Przygotuj dane projektu `pakowanie-paczek` dla wspólnego silnika aplikacji Android.

## MVP

Użytkownik wybiera typ przedmiotu, kruchość, rozmiar i sposób wysyłki. Aplikacja zwraca instrukcję pakowania.

## Dane do przygotowania

- typy przedmiotów
- poziomy kruchości
- rozmiary paczek
- materiały zabezpieczające
- instrukcje krok po kroku
- checklisty przed nadaniem

## Zasady

Nie gwarantuj pełnej ochrony. Dla drogich lub delikatnych rzeczy dodaj ostrzeżenie o ubezpieczeniu przesyłki.
