# Fryzury Proste

## Cel aplikacji

Aplikacja pokazuje neutralne instrukcje prostych uczesań krok po kroku.

## Schemat MVP

rodzaj włosów -> czas -> okazja -> instrukcja podstawowa -> reklama -> dodatkowe warianty

## Zakres MVP

- uczesania szybkie
- uczesania sportowe
- uczesania codzienne
- uczesania eleganckie
- lista potrzebnych akcesoriów
- instrukcje krok po kroku

## Wynik

Aplikacja pokazuje:

- nazwę uczesania
- potrzebne akcesoria
- kroki wykonania
- wariant szybszy
- wariant staranniejszy

## Monetyzacja

- wybrane instrukcje za darmo
- kolejne warianty po reklamie
- paczki tematyczne po reklamie

## Zasady

Aplikacja nie ocenia osoby, twarzy ani atrakcyjności. Pokazuje wyłącznie neutralne instrukcje wykonania uczesania.
