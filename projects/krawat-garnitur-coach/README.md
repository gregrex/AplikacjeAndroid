# Krawat Garnitur Coach

## Cel aplikacji

Aplikacja pomaga dobrać prosty formalny zestaw: garnitur, koszulę, krawat lub muchę, buty i pasek.

## Schemat MVP

okazja -> kolor garnituru -> kolor koszuli -> dodatki -> wynik podstawowy -> reklama -> pełna checklista

## Okazje startowe

- rozmowa
- uroczystość rodzinna
- egzamin
- spotkanie biznesowe
- eleganckie wyjście

## Wynik

- bezpieczny zestaw
- czego unikać
- dobór dodatków
- checklista przed wyjściem

## Monetyzacja

- jedna rekomendacja za darmo
- warianty po reklamie
- checklista po reklamie
- instrukcja wiązania po reklamie

## Zasady

Aplikacja nie ocenia osoby. Ocenia wyłącznie zgodność elementów stroju z okazją.
