# Prompt dla Codexa — Krawat Garnitur Coach

## Cel

Przygotuj dane projektu `krawat-garnitur-coach` dla wspólnego silnika aplikacji Android.

## MVP

Użytkownik wybiera okazję, kolor garnituru, kolor koszuli i dodatki. Aplikacja tworzy bezpieczną rekomendację formalnego zestawu.

## Dane do przygotowania

- okazje
- kolory garniturów
- kolory koszul
- krawat lub mucha
- buty i pasek
- checklisty formalności
- reguły czego unikać

## Zasady

Nie oceniaj wyglądu osoby. Projekt dotyczy dopasowania elementów ubioru do okazji.
