# Router WiFi Diagnosta

## Cel aplikacji

Aplikacja pomaga poprawić zasięg i stabilność domowej sieci WiFi na podstawie prostego formularza.

## Schemat MVP

problem -> typ mieszkania -> miejsce routera -> liczba ścian -> wynik podstawowy -> reklama -> pełny plan poprawy

## Problemy startowe

- słaby zasięg
- niska prędkość
- zrywanie połączenia
- problem w jednym pokoju
- problem z wieloma urządzeniami

## Wynik

- pierwsza rzecz do sprawdzenia
- propozycja przestawienia routera
- checklista ustawień
- kiedy warto kupić mesh lub repeater

## Monetyzacja

- pierwsza diagnoza za darmo
- pełny plan po reklamie
- mapa testów po mieszkaniu po reklamie

## Zasady

Aplikacja nie zmienia ustawień routera automatycznie. Daje instrukcje i checklisty do samodzielnego sprawdzenia.
