# Prompt dla Codexa — Router WiFi Diagnosta

## Cel

Przygotuj dane projektu `router-wifi-diagnosta` dla wspólnego silnika aplikacji Android.

## MVP

Użytkownik opisuje problem z WiFi, typ mieszkania i ustawienie routera. Aplikacja pokazuje plan poprawy.

## Dane do przygotowania

- typy problemów
- pytania o mieszkanie
- pytania o router
- checklisty ustawień
- rekomendacje rozmieszczenia
- decyzje: router, repeater, mesh, kabel

## Zasady

Nie wykonuj automatycznych zmian sieci. Nie obiecuj gwarantowanej poprawy. Daj praktyczne kroki testowe.
