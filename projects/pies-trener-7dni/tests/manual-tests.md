# Testy manualne — Pies Trener 7 Dni

## 1. Szczeniak, łatwy poziom

Wybierz:

- wiek: szczeniak,
- energia: średnia,
- poziom: łatwy.

Oczekiwane:

- wynik `puppy_easy`,
- plan 7 dni,
- ostrzeżenie, aby nie przeciążać psa.

## 2. Wysoka energia

Wybierz:

- pies dorosły,
- energia: wysoka,
- poziom: normalny.

Oczekiwane:

- wynik `high_energy`,
- zalecenie wyciszenia przed treningiem.

## 3. Senior

Wybierz:

- wiek: senior,
- energia: niska.

Oczekiwane:

- wynik spokojny,
- komunikat o obserwowaniu dyskomfortu.

## 4. Trudny poziom

Wybierz:

- poziom: trudny.

Oczekiwane:

- aplikacja ostrzega, że poważne problemy wymagają specjalisty.

## 5. Ulubione i historia

Po pokazaniu wyniku:

- dodaj wynik do ulubionych,
- sprawdź historię.

Oczekiwane:

- wynik zapisuje się lokalnie,
- historia zawiera projekt i wynik.
