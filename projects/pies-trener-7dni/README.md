# Pies Trener 7 Dni

## Cel aplikacji

Aplikacja daje prosty plan treningowy dla psa dzień po dniu.

## Schemat MVP

```text
wiek psa
→ cel treningu
→ poziom trudności
→ plan podstawowy
→ reklama rewarded
→ pełny plan 7 dni
```

## Zakres MVP

Komendy i cele:

- siad,
- leżeć,
- zostań,
- do mnie,
- łapa,
- nie skacz,
- nie ciągnij na smyczy,
- zostaw.

## Bezpieczeństwo

Aplikacja nie zastępuje behawiorysty ani weterynarza. Przy agresji, silnym lęku lub nagłej zmianie zachowania należy skonsultować się ze specjalistą.

## Monetyzacja

- darmowy dzień 1,
- kolejne dni po reklamie,
- paczki trików premium,
- wersja bez reklam.
