# Szydełko Pomocnik

## Cel aplikacji

Aplikacja pomaga w prostym prowadzeniu projektów szydełkowych i korzystaniu z podstawowych instrukcji.

## Schemat MVP

rodzaj projektu -> poziom trudności -> licznik rzędów -> porada podstawowa -> reklama -> dodatkowe wzory lub instrukcje

## Funkcje MVP

- licznik rzędów
- notatki projektu
- słownik skrótów
- proste ściegi
- lista materiałów
- proste wzory

## Monetyzacja

- podstawowe narzędzia za darmo
- dodatkowe wzory po reklamie
- eksport projektu po reklamie
- wersja bez reklam

## Zasady

Aplikacja działa offline i nie wymaga AI w MVP.
