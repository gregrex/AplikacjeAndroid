# Prompt dla Codexa — Bukietownik

## Cel

Przygotuj dane projektu `bukietownik` dla wspólnego silnika aplikacji Android.

## MVP

Użytkownik wybiera okazję, kolory, dostępne kwiaty i styl. Aplikacja proponuje prostą kompozycję.

## Dane do przygotowania

- okazje
- style bukietów
- kolory
- kwiaty
- dodatki
- instrukcje układania
- porady pielęgnacyjne

## Zasady

Nie obiecuj trwałości bukietu. Daj praktyczne, proste porady.
