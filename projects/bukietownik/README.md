# Bukietownik

## Cel aplikacji

Aplikacja pomaga dobrać prostą kompozycję kwiatową do okazji, kolorów i dostępnych kwiatów.

## Schemat MVP

okazja -> kolory -> kwiaty -> styl -> propozycja podstawowa -> reklama -> warianty bukietu

## Okazje startowe

- urodziny
- podziękowanie
- spotkanie rodzinne
- dekoracja domu
- prezent bez okazji

## Wynik

- propozycja kompozycji
- kolejność układania
- dodatki
- pielęgnacja bukietu

## Monetyzacja

- jedna propozycja za darmo
- warianty po reklamie
- opis do social media po reklamie
- paczki stylów
