# Kot Bawi się

## Cel aplikacji

Aplikacja dobiera proste zabawy i aktywności dla kota na podstawie wieku, poziomu energii i czasu właściciela.

## Schemat MVP

wiek kota -> energia -> czas -> zabawa podstawowa -> reklama -> plan dnia lub tygodnia

## Zakres MVP

Typy zabaw:

- zabawy krótkie
- zabawy z wędką
- zabawy z kartonem
- zabawy DIY
- zabawy dla kota niewychodzącego
- zabawy dla seniora

## Monetyzacja

- kilka zabaw dziennie za darmo
- dodatkowe zabawy po reklamie
- plan tygodnia po reklamie
- wersja bez reklam

## Ograniczenia

Aplikacja nie diagnozuje zdrowia kota. Przy nagłej zmianie zachowania, bólu, braku apetytu lub problemach z kuwetą należy skonsultować się z weterynarzem.
