# Prompt dla Codexa — Kot Bawi się

## Cel

Przygotuj dane projektu `kot-bawi-sie` dla wspólnego silnika aplikacji Android.

## MVP

Aplikacja dobiera zabawy dla kota według wieku, energii, miejsca i czasu.

## Dane do przygotowania

- wiek kota
- poziom energii
- czas właściciela
- typ mieszkania
- lista zabaw
- plan tygodnia
- ostrzeżenia bezpieczeństwa

## Zasady

Nie diagnozuj zdrowia. Aplikacja ma być poradnikiem zabawy, aktywności i wzbogacenia środowiska.
