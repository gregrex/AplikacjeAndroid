# Prompt dla Codexa — Silikon Fuga Fix

## Cel

Przygotuj dane projektu `silikon-fuga-fix` dla wspólnego silnika aplikacji Android.

## MVP

Użytkownik wybiera problem, miejsce i stan zabrudzenia. Aplikacja pokazuje wskazówkę lub pełną instrukcję.

## Dane do przygotowania

- problemy fug
- problemy silikonu
- miejsca użycia
- narzędzia
- środki czyszczące
- instrukcje wymiany
- ostrzeżenia kiedy przerwać

## Zasady

Nie udawaj diagnozy budowlanej. Przy poważnym przecieku lub wilgoci w ścianie kieruj użytkownika do fachowca.
