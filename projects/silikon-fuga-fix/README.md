# Silikon Fuga Fix

## Cel aplikacji

Aplikacja pomaga dobrać prostą procedurę czyszczenia, odświeżenia albo wymiany silikonu i fug.

## Schemat MVP

problem -> miejsce -> stan zabrudzenia -> materiał -> wskazówka podstawowa -> reklama -> pełna instrukcja

## Problemy startowe

- zabrudzona fuga
- stary silikon
- pleśń na silikonie
- pęknięcie
- odspojenie
- przeciek podejrzany

## Wynik

- czy czyścić czy wymienić
- lista narzędzi
- kroki przygotowania
- pełna instrukcja
- kiedy wezwać fachowca

## Monetyzacja

- podstawowa wskazówka za darmo
- pełna instrukcja po reklamie
- lista zakupów po reklamie

## Zasady bezpieczeństwa

Przy podejrzeniu większego przecieku, uszkodzenia instalacji lub wilgoci w ścianie aplikacja ma zalecać konsultację z fachowcem.
