# DomFix

## Cel aplikacji

Aplikacja pomaga użytkownikowi wykonać drobne, bezpieczne naprawy domowe albo zdecydować, kiedy lepiej wezwać fachowca.

## Schemat MVP

problem -> miejsce -> narzędzia -> poziom ryzyka -> instrukcja podstawowa -> reklama -> pełna instrukcja

## Zakres MVP

Problemy startowe:

- skrzypiące drzwi
- luźny zawias
- drobna rysa na meblu
- zatkany odpływ podstawowy
- wymiana silikonu
- fuga do odświeżenia
- zamek błyskawiczny
- drobne uszczelnienie

## Wynik

- pierwszy bezpieczny krok
- lista narzędzi
- pełna instrukcja
- kiedy przerwać i wezwać fachowca

## Monetyzacja

- podstawowy krok za darmo
- pełna instrukcja po reklamie
- lista zakupów po reklamie

## Zasady bezpieczeństwa

Nie prowadzić użytkownika przez niebezpieczne prace elektryczne, gazowe, konstrukcyjne ani hydrauliczne wysokiego ryzyka.
