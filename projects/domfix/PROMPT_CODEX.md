# Prompt dla Codexa — DomFix

## Cel

Przygotuj dane projektu `domfix` dla wspólnego silnika aplikacji Android.

## MVP

Użytkownik wybiera problem domowy, miejsce, dostępne narzędzia i poziom ryzyka. Aplikacja pokazuje instrukcję albo zalecenie wezwania fachowca.

## Dane do przygotowania

- lista problemów
- poziomy ryzyka
- lista narzędzi
- instrukcje krok po kroku
- ostrzeżenia bezpieczeństwa
- reguły przerwania pracy

## Zasady

Nie twórz porad do prac wysokiego ryzyka. Nie zachęcaj do napraw instalacji gazowych, elektrycznych i konstrukcyjnych. Przy ryzyku pokazuj komunikat o fachowcu.
