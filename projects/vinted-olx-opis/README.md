# Vinted OLX Opis

## Cel aplikacji

Aplikacja pomaga szybko przygotować opis ogłoszenia sprzedażowego na podstawie formularza.

## Schemat MVP

kategoria przedmiotu -> stan -> cechy -> wady -> forma odbioru -> opis podstawowy -> reklama -> warianty opisów

## Zakres danych

Kategorie startowe:

- ubrania
- buty
- zabawki
- książki
- elektronika
- meble
- akcesoria

## Wynik

Aplikacja generuje:

- tytuł ogłoszenia
- krótki opis
- pełny opis
- listę brakujących zdjęć
- hasła pomocnicze

## Monetyzacja

- jeden opis podstawowy za darmo
- dodatkowe warianty po reklamie
- checklisty zdjęć po reklamie
- opcjonalna wersja bez reklam

## Ryzyka

Nie zaczynać od automatycznej wyceny. Wycena może być później i tylko jako orientacyjna sugestia.
