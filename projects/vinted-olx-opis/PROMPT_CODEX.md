# Prompt dla Codexa — Vinted OLX Opis

## Cel

Przygotuj projekt danych `vinted-olx-opis` dla wspólnego silnika aplikacji Android.

## MVP

Użytkownik wybiera kategorię, stan, kolor, rozmiar lub cechy oraz ewentualne wady. Aplikacja tworzy opis ogłoszenia.

## Dane do przygotowania

- kategorie przedmiotów
- pytania o stan
- pytania o cechy
- pytania o wady
- szablony tytułów
- szablony opisów
- checklisty zdjęć

## Zasady

Nie generuj fałszywych informacji. Jeżeli użytkownik zaznaczy wadę, opis ma ją jasno uwzględnić. Nie obiecuj automatycznej wyceny w MVP.
