# Kołek Dobieracz

## Cel aplikacji

Aplikacja pomaga dobrać kołek, śrubę i wiertło do prostego montażu domowego.

## Schemat MVP

```text
co montujesz
→ typ ściany
→ przybliżona waga
→ wynik podstawowy
→ reklama rewarded
→ pełna instrukcja montażu
```

## Zakres MVP

Kategorie montażu:

- obraz,
- półka,
- lustro,
- karnisz,
- haczyk,
- listwa,
- mała szafka.

Typy ścian:

- beton,
- cegła,
- karton-gips,
- drewno,
- nie wiem.

## Ostrzeżenia

Aplikacja nie zastępuje fachowca. Przy ciężkich elementach, TV, szafkach wiszących i montażu nad głową użytkownik powinien skonsultować się z fachowcem.

## Monetyzacja

- darmowa rekomendacja podstawowa,
- pełna instrukcja po reklamie,
- lista narzędzi po reklamie lub w premium.
