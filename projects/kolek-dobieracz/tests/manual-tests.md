# Testy manualne — Kołek Dobieracz

## 1. Start projektu

Otwórz aplikację i wybierz projekt `kolek-dobieracz`.

Oczekiwane:

- ekran startowy pokazuje nazwę projektu,
- styl jest techniczny i narzędziowy,
- aplikacja pokazuje kategorie montażu.

## 2. Lekki obraz na betonie

Wybierz:

- kategoria: obraz,
- ściana: beton,
- waga: lekka.

Oczekiwane:

- wynik podstawowy: beton, lekki przedmiot,
- po odblokowaniu wynik pełny z listą narzędzi.

## 3. Średni przedmiot na betonie

Wybierz:

- kategoria: półka,
- ściana: beton,
- waga: średnia.

Oczekiwane:

- aplikacja sugeruje mocniejsze mocowanie,
- pokazuje ostrzeżenie o zapasie nośności.

## 4. Karton-gips lekki

Wybierz:

- ściana: karton-gips,
- waga: lekka.

Oczekiwane:

- wynik wskazuje kołek do płyt g-k,
- ostrzega o ograniczonej nośności.

## 5. Ciężki montaż

Wybierz:

- dowolna ściana,
- waga: ciężka.

Oczekiwane:

- aplikacja nie daje zbyt pewnej instrukcji,
- wynik kieruje do oceny nośności lub fachowca.

## 6. Historia i ulubione

Po pokazaniu wyniku:

- dodaj do ulubionych,
- wróć do historii.

Oczekiwane:

- wpis pojawia się w historii,
- wynik można zapisać jako ulubiony.
