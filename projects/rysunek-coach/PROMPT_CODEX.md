# Prompt dla Codexa — Rysunek Coach

## Cel

Przygotuj dane projektu `rysunek-coach` dla wspólnego silnika aplikacji Android.

## MVP

Aplikacja pokazuje lekcje rysowania krok po kroku według tematu i poziomu trudności.

## Dane do przygotowania

- tematy lekcji
- poziomy trudności
- kroki instrukcji
- materiały potrzebne
- ćwiczenia dzienne
- paczki tematyczne

## Zasady

Nie używaj cudzych marek ani postaci. Lekcje mają być autorskie i neutralne.
