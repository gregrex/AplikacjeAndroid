# Rysunek Coach

## Cel aplikacji

Aplikacja pokazuje proste lekcje rysowania krok po kroku.

## Schemat MVP

temat -> poziom trudności -> lekcja podstawowa -> reklama -> dodatkowe kroki lub karta ćwiczeń

## Kategorie startowe

- zwierzęta
- pojazdy
- rośliny
- postacie proste
- roboty
- domy
- potwory

## Monetyzacja

- kilka lekcji za darmo
- kolejne lekcje po reklamie
- paczki tematyczne po reklamie
- eksport karty ćwiczeń po reklamie

## Zasady

Nie używać nazw ani postaci chronionych prawem autorskim. Tworzyć własne neutralne tematy.
