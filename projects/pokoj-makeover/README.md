# Pokój Makeover

## Cel aplikacji

Aplikacja pomaga zaplanować prostą metamorfozę pokoju bez remontu i bez dużego budżetu.

## Schemat MVP

typ pokoju -> styl -> budżet -> problem -> plan podstawowy -> reklama -> pełny plan zmian

## Zakres MVP

Style:

- minimalistyczny
- przytulny
- kolorowy
- gaming
- nauka i praca
- mały pokój

## Wynik

Aplikacja podaje:

- 3 szybkie zmiany
- plan sprzątania i ustawienia rzeczy
- pomysły dekoracji
- listę zakupów lub DIY

## Monetyzacja

- 3 pomysły za darmo
- pełny plan po reklamie
- wariant budżetowy po reklamie

## Zasady

Aplikacja nie analizuje zdjęcia w MVP. Użytkownik opisuje pokój przez formularz.
