# Prompt dla Codexa — Outfit Coach

## Cel

Przygotuj dane projektu `outfit-coach` dla wspólnego silnika aplikacji Android.

## MVP

Użytkownik wybiera okazję, pogodę, preferowany styl i dostępne elementy garderoby. Aplikacja tworzy prostą propozycję zestawu.

## Dane do przygotowania

- okazje
- pogoda
- style
- elementy ubioru
- reguły dopasowania formalności
- reguły kolorów podstawowych
- listy ostrzeżeń czego unikać

## Zasady

Nie oceniaj ciała użytkownika. Nie używaj rankingów atrakcyjności. Porady mają dotyczyć ubrań, okazji, kolorów i praktyczności.
