# Outfit Coach

## Cel aplikacji

Aplikacja pomaga dobrać prosty zestaw ubrań do okazji, pogody i stylu.

## Schemat MVP

okazja -> pogoda -> styl -> elementy garderoby -> propozycja podstawowa -> reklama -> warianty

## Zakres MVP

Okazje:

- szkoła lub uczelnia
- spotkanie
- rozmowa
- impreza
- spacer
- uroczystość rodzinna

## Wynik

Aplikacja podaje:

- proponowany zestaw
- czego unikać
- wariant tańszy
- lista kontrolna przed wyjściem

## Zasady

Aplikacja ocenia wyłącznie dopasowanie ubrań do okazji. Nie ocenia ciała, wagi, atrakcyjności ani osoby.

## Monetyzacja

- jedna propozycja za darmo
- dodatkowe warianty po reklamie
- lista zakupów po reklamie
