# Zmywarka Diagnosta

## Cel aplikacji

Aplikacja pomaga użytkownikowi przejść prostą checklistę typowych problemów ze zmywarką.

## Schemat MVP

objaw -> częstotliwość -> ostatnie zmiany -> pierwszy krok -> reklama -> pełna checklista

## Objawy startowe

- nie domywa
- zostawia osad
- brzydko pachnie
- tabletka się nie rozpuszcza
- naczynia są mokre
- nie odpompowuje wody
- pokazuje błąd

## Wynik

- bezpieczny pierwszy krok
- kolejność sprawdzania
- czego nie robić
- kiedy wezwać serwis

## Monetyzacja

- pierwszy krok za darmo
- pełna checklista po reklamie
- lista możliwych przyczyn po reklamie

## Zasady bezpieczeństwa

Nie prowadzić przez naprawy elektryczne. Przy wycieku, zapachu spalenizny lub ryzyku porażenia zalecić odłączenie urządzenia i kontakt z serwisem.
