# Chleb Zakwas Coach

## Cel aplikacji

Aplikacja pomaga prowadzić zakwas i rozwiązywać podstawowe problemy przy pieczeniu chleba.

## Schemat MVP

problem -> etap -> temperatura -> czas -> wskazówka podstawowa -> reklama -> pełny plan ratunkowy

## Zakres MVP

Problemy:

- zakwas nie rośnie
- ciasto nie wyrasta
- chleb ma zakalec
- skórka jest blada
- chleb pęka
- zakwas ma dziwny zapach

## Wynik

- możliwa przyczyna
- pierwszy krok
- pełny plan korekty
- harmonogram karmienia

## Monetyzacja

- wskazówka podstawowa za darmo
- pełny plan po reklamie
- harmonogram po reklamie

## Zasady

Przy podejrzeniu pleśni lub zepsucia aplikacja powinna zalecić wyrzucenie zakwasu, a nie ratowanie za wszelką cenę.
